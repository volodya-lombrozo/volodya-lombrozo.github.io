I will be happy to correct them.
You can:

1) Email me on [volodya.lombrozo@gmail.com](volodya.lombrozo@gmail.com)

3) Or even create
   a [pull request](https://github.com/volodya-lombrozo/volodya-lombrozo.github.io/pulls)
   with the correction.
